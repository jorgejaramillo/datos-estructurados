¿Qué son los datos estructurados de Google y para qué sirven?

Los datos estructurados se podrían catalogar como una convención entre varios grandes de la web. Es muy fácil para nosotros como humanos reconocer las fotos de una receta, el tiempo de preparación y las calorías. Pero no es tan fácil para los motores de búsqueda catalogar cada uno de estos ítems. Los datos estructurados funcionan para páginas de recetas, como mencioné, pero también para un gran número de tipologías de contenido.

Páginas de recetas

Si tienes una página, donde publiques recetas, este conjunto de datos estructurados te pueden ayudar a mejorar la visibilidad de tu sitio.

Página de Preguntas frecuentes

Las famosas FAQs o páginas con las preguntas comunes en nuestro negocio. Antes, esa página era plana de cara a Google. Ahora podemos lograr un bloque con interacción dentro de la página de resultados o SERP. Este es uno de los Datos Estructurados nuevos y con mayor crecimiento en la actualidad.

Miga de pan

Los breadcrumbs sirven para dar una jerarquía dentro de un sitio web. Por medio de este formato de datos estructurados podemos mejorar la visibilidad de la URL en Google. Le da un toque más profesional a nuestro resultado.

Productos en Ecommerce

Ideal para tiendas en linea, las cuales buscan una visibilidad mejor para sus productos en Google. Esta opción permite poner la foto visible, también el precio y otros atributos propios de este tipo de contenido.



La recomendación de Google es utilizar estos bloques de datos en el formato JSON. También existen otros métodos como los microformatos y RDFa, sin embargo, pueden ser ineficientes.
